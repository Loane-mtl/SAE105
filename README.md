# URL site web - Mon Univers Sims 
Le site est accessible à l'adresse suivante : http://martinel.projetsmmichamps.fr/


## Installer les fichiers de l'archive sur un serveur local XAMPP.

1.Ouvrir XAMPP
- Lancer XAMPP
- Démarrer Apache 

2.Mettre site dans le bon dossier 
- Aller dans XAMPP puis htdocs 
- Copier le dossier du projet (SAE-105-TPA-Loane-Martinel)
- Le coller dans htdocs 

3.Ouvrir le site dans le navigateur 
- Dans la barre du navigateur, écrire : http://localhost/SAE-105-TPA-Loane-Martinel/
- Le site s'affichera.
